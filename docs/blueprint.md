# **App Name**: FoodNexus

## Core Features:

- Browse Meals: Allow users to browse meals available from local vendors, restaurant etc. in Calabar.
- Filter Meals: Enable users to filter meals based on category, price.
- Vendor Price Comparison: Display prices when search or scrolled to, from various vendors for the same or similar meals, facilitating comparison.
- Meal Ordering: Allow users to place orders for meals directly through the app.

## Style Guidelines:

- Primary color: Sunny yellow (#FFD700) to evoke warmth and appetite.
- Background color: Desaturated yellow (#FAF5E6), creating a soft and inviting base.
- Accent color: Earthy brown (#A0522D) for a natural and grounded feel.
- Body and headline font: 'PT Sans' for a modern and accessible feel.
- Use food-related icons with a line art style to complement the color palette.
- Grid-based layout for meal browsing, ensuring easy navigation.
- Subtle transition animations when filtering and ordering meals.
